# Winter to Spring2026 Personal Practice Project

## Functional Features
-Responsive design with mobile device support

-Clean and aesthetically pleasing UI interface

-Pure static pages, no backend support required

-Integrates Windows 11 rounded corner design and iOS 26 glass texture design

## Project Introduction
This project was initiated in September 2023, with version 1.0.0 released in 2024. The 2026 project started development on December 4, 2025, aiming to showcase personal annual key events and convey heartfelt messages to specific individuals on the occasion of the New Year.
